# ブログ投稿の削除処理実装

## ブログ投稿削除関連のルーティング追加
### 「blog/routes」配下にある、web.phpに下記を追加する。

    Route::delete('/posts/{post}', [PostController::class,'delete']);

## ブログ投稿削除実行用のコントローラー実装
### 「blog/app/Http/Controllers」配下にある、PostController.phpに下記を追加する。

    public function delete(Post $post)
    {
        $post->delete();
        return redirect('/');
    }
