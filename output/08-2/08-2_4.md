# ブログ投稿一覧のページネーション追加

### Modelクラス（Post.php）に下記を追加
※ 前の項目で記述していたgetByLimitは一旦コメントアウト

    public function getPaginateByLimit(int $limit_count = 10)
    {
        // updated_atで降順に並べたあと、limitで件数制限をかける
        return $this->orderBy('updated_at', 'DESC')->paginate($limit_count);
    }
